### Our Questions

Before building anything:

1. **Can we fake this manually first?**
2. **What's the smallest version that delivers value?**
3. **Can we ship this by Thursday?**
4. **Who will demo this to the client?**

Before any debate lasting >15 minutes:

5. **What's the simplest thing that works?**
6. **Can we change this later if we're wrong?**
