## Our Core Philosophy

**We ship working software every Thursday.**

Why? Because:
- Real feedback beats assumptions
- Deployed code teaches us more than planning
- Fast iteration is our competitive advantage
- Companies ship, perfect systems don't exist

Read the full [Ship First Manifesto](./01-ship-first-manifesto.md) to understand our culture.

## Our Weekly Cadence

| Day | What Happens |
|-----|--------------|
| **Monday** | Plan: What ships Thursday? |
| **Tue-Thu** | Build: Merge daily, pair on integration |
| **Thursday** | Deploy: Ship to production (4 PM) |
| **Friday** | Enable: Turn on features & demo |
